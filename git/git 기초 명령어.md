# git 기초 명령어

| 명령어                      | 내용                                   |
| --------------------------- | -------------------------------------- |
| git init                    | 로컬 저장소 생성                       |
| git add '파일명'            | 특정 파일/폴더의 변경사항 추가         |
| git commit -m '커밋 메시지' | commit(버전 기록)                      |
| git status                  | git의 상태를 확인 빨강과 초록으로 구분 |
| git log                     | commit(버전) 기록 확인                 |

4. 데이터 구조 & 알고리즘

   > 알고 푸는 것과 모르고 푸는 것은 천지 차이.

   - 프로그램 = 데이터 구조 + 알고리즘 Nikalus Wirth

     어떻게 저장할 것인가 -> 데이터 구조

     어떻게  계산할 것인가 -> 알고리즘

     - 데이터 구조

       데이터를 다양한 방식으로 저장하고 조회, 삽입, 변경, 삭제와 같은 조작 기능을 제공한다.

       💡 **왜 데이터 구조가 중요할까?** a.k.a 데이터(물) + 구조(통) == 데이터 구조(물통)

       문제상황에 따라 <u>더 적합한 도구</u>가 필요하다!!! **문제를 더 효율적으로 풀기 위한 도구**

# CSS Display

>  모든 요소는 박스 모델이고 css display에 따라 달라진다.

- display : block

  줄 바꿈이 일어나는 요소

  막으니까 떨어진다 막고 있으니까 떨어진다

- display : inline

  한 줄에, 같은 줄 안에 있다

  줄바꿈이 일어나지 않는다

  content 너비만큼 가로 폭을 차지한다
